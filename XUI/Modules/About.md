### About 关于


此组件在界面上显示一个关于区域.


|   键   |   类型   |   描述   |   条件   |
|--------|----------|----------|----------|
|imagePath|字符串|关于图标|可选|


此组件需要设定通用属性 **icon**/**label**/**value**, 分别显示为图标/标题/副标题. 


```lua
{
    cell = "About";
    label = "XXTouch\nv1.2.0-1";
    value = "https://www.xxtouch.com\n2016-2018 (c) XXTouch Team.\nAll Rights Reserved.";
};
```
