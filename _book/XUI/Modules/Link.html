Link 链接子界面

此组件在界面上显示一个子菜单项, 用于链接子界面.

键 类型 描述 条件
url 字符串 子界面文件名 -

url 可以为普通文件名、XUI 文件名或网络地址. 普通文件将使用默认打开方式打开, XUI 文件将作为子界面打开, 网络地址将使用内置浏览器打开. 

{
    url = "sub/xui-sub.xui";
    cell = "Link";
    label = "Load another pane";
};
